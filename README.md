# Rule-Based Chatbot

## Project Overview

The Rule-Based Chatbot is a simple conversational AI application developed in Python. It responds to user queries using predefined rules and pattern matching instead of machine learning algorithms. This project demonstrates the fundamentals of Natural Language Processing (NLP), conditional statements, and conversation flow.

---

## Features

- Greets the user
- Answers common questions
- Displays the current date
- Displays the current time
- Performs simple mathematical calculations
- Shares motivational quotes
- Tells programming jokes
- Answers basic general knowledge questions
- Handles unknown inputs gracefully
- Exits the conversation when requested

## Installation

1. Clone the repository

```
git clone https://github.com/yourusername/Rule-Based-Chatbot.git
```

2. Open the project folder

```
cd Rule-Based-Chatbot
```

3. Run the chatbot

```
python chatbot.py
```
